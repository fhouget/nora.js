#nora.js#

SOAP and REST Web Services Non-Regression Automation with Javascript

## Depencencies
 - python
 - npm
 - node.js

## Install
$ npm install

## Run the getWeather sample testcase
$ npm test

## Run my own testcase
$ TESTCASE=./<testcase>.json npm start
